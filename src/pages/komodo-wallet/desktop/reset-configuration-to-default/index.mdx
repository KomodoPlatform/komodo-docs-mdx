# Resetting Komodo Wallet Configuration to Default

Sometimes an improper shut down may result in your coins.json file being corrupted. If you load Komodo Wallet and see no coins available, follow the steps below:

1.  Click on settings in the lower part of the sidebar.

<OptimizedImage title="Resetting Komodo Wallet Configuration to Default" src={resetConfigurationToDefault1} classNaming="w-200" alt="Resetting Komodo Wallet Configuration to Default" />

2.  Under the General tab, on the line for **Reset wallet configuration**, click on the Reset button.

<OptimizedImage title="Resetting Komodo Wallet Configuration to Default" src={resetConfigurationToDefault2} classNaming="w-full" alt="Resetting Komodo Wallet Configuration to Default" />

3.  Click confirm to reset your configuration.

<OptimizedImage title="Resetting Komodo Wallet Configuration to Default" src={resetConfigurationToDefault3} classNaming="w-300" alt="Resetting Komodo Wallet Configuration to Default" />

4.  This will restart the app.

<OptimizedImage title="Resetting Komodo Wallet Configuration to Default" src={resetConfigurationToDefault4} classNaming="w-300" alt="Resetting Komodo Wallet Configuration to Default" />

When you log in again, the default coins should be enabled and visible. To activate other coins, refer to the [Komodo Wallet Desktop coin activation guide](https://forum.komodoplatform.com/t/activating-coins-in-atomicdex-desktop/571).
