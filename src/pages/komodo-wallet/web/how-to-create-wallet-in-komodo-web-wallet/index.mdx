# How to Create Wallet in Komodo Web Wallet

*   In a new browser tab, open [Komodo Web Wallet](https://app.komodoplatform.com/#/wallet)

*   Click the “Connect wallet” button on the top of the page:

<OptimizedImage title=" How to Create Wallet in Komodo Web Wallet" src={howToCreateWallet1} alt=" How to Create Wallet in Komodo Web Wallet" />;

*   Click the “Connect Komodo Wallet seed” button

<OptimizedImage title=" How to Create Wallet in Komodo Web Wallet" src={howToCreateWallet2} alt=" How to Create Wallet in Komodo Web Wallet" />;

*   Click the “Create wallet” button

<OptimizedImage title=" How to Create Wallet in Komodo Web Wallet" src={howToCreateWallet3} alt=" How to Create Wallet in Komodo Web Wallet" />;

*   You need to create a name for your wallet and set a password. To confirm you have read the the **End-User License Agreement (EULA)** and **TERMS and CONDITIONS**, accept them by activating their checkboxes to continue.

*   Once all inputs have been validated, click the “Create” button to create your wallet

<OptimizedImage title=" How to Create Wallet in Komodo Web Wallet" src={howToCreateWallet4} alt=" How to Create Wallet in Komodo Web Wallet" />;

*   Voila! Your wallet has been created!

<OptimizedImage title=" How to Create Wallet in Komodo Web Wallet" src={howToCreateWallet5} alt=" How to Create Wallet in Komodo Web Wallet" />;

> **IMPORTANT!** You need to BACKUP your seed phrase before making a transaction with non-test coins. Please save your seed phrase offline and keep it in a safe place!
