# How to Deactivate Assets in Komodo Web Wallet

*   In a new browser tab, open [Komodo Wallet Web Version](https://app.komodoplatform.com/#/dex) in your browser and [CREATE](/komodo-wallet/web/how-to-deactivate-assets-in-komodo-web-wallet/) or [IMPORT](/komodo-wallet/web/how-to-deactivate-assets-in-komodo-web-wallet/) your wallet

*   Open the Wallet tab and click the “Remove assets” button.

<OptimizedImage title=" How to Deactivate Assets in Komodo Web Wallet" src={howToDeactivate1} alt=" How to Deactivate Assets in Komodo Web Wallet" />

*   You will see the full list of your activated assets and balances.

<OptimizedImage title=" How to Deactivate Assets in Komodo Web Wallet" src={howToDeactivate2} alt=" How to Deactivate Assets in Komodo Web Wallet" />

*   Use filters to see only the Protocols you are interested in, or use the searchbar to filter by name or ticker.

<OptimizedImage title=" How to Deactivate Assets in Komodo Web Wallet" src={howToDeactivate3} alt=" How to Deactivate Assets in Komodo Web Wallet" />

*   Select all the assets you want to deactivate, then click the “Remove assets” button.
    *   **NOTE:** you cannot deactivate default assets (KMD, LTC and BTC) or assets which have a swap in progress.

<OptimizedImage title=" How to Deactivate Assets in Komodo Web Wallet" src={howToDeactivate4} alt=" How to Deactivate Assets in Komodo Web Wallet" />

*   Selected assets are now removed from your wallet page :slight\_smile: You can always [activate](/komodo-wallet/web/how-to-deactivate-assets-in-komodo-web-wallet/) any asset again later.
