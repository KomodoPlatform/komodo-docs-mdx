# How to Activate Assets in Komodo Web Wallet

*   In a new browser tab, open [Komodo Wallet Web Version](https://app.komodoplatform.com/#/dex) in your browser and [CREATE](/komodo-wallet/web/how-to-activate-assets-in-komodo-web-wallet/) or [IMPORT](/komodo-wallet/web/how-to-activate-assets-in-komodo-web-wallet/) your wallet

*   Open the Wallet tab and click the “Add assets” button.

<OptimizedImage title=" How to Activate Assets in Komodo Web Wallet" src={howToActivate1} alt=" How to Activate Assets in Komodo Web Wallet" />

*   You will see the complete list of available assets on Komodo Web Wallet.

<OptimizedImage title=" How to Activate Assets in Komodo Web Wallet" src={howToActivate2} alt=" How to Activate Assets in Komodo Web Wallet" />

*   Use filters to see only the Protocols you are interested in.

<OptimizedImage title=" How to Activate Assets in Komodo Web Wallet" src={howToActivate3} alt=" How to Activate Assets in Komodo Web Wallet" />

*   Use the searchbar to filter by name or ticker.

<OptimizedImage title=" How to Activate Assets in Komodo Web Wallet" src={howToActivate4} alt=" How to Activate Assets in Komodo Web Wallet" />

*   Click the “Add assets” button to activate the selected coins. After clicking the “Add assets” button, you will see a “loading” spinner for a short period before being returned to the Wallet page.
    *   Once activation has completed, you will see your selected coins and balances on the wallet page.
    *   All activated coins will automatically be enabled again when logging this wallet on the same device until you deactivate them.
