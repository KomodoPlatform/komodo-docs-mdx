# How to Import Wallet in Komodo Web Wallet

*   In a new browser tab, open [Komodo Web Wallet](https://app.komodoplatform.com/#/wallet)

*   Click the “Connect wallet” button on the top of the page:

<OptimizedImage title=" How to Import Wallet in Komodo Web Wallet" src={howToImportWallet1} alt=" How to Import Wallet in Komodo Web Wallet" />;

*   Alternatively, you can click on any asset from the assets list or the “Connect wallet” button on the DEX or BRIDGE tab.

*   Click the “Connect Komodo Wallet seed” button

<OptimizedImage title=" How to Import Wallet in Komodo Web Wallet" src={howToImportWallet2} alt=" How to Import Wallet in Komodo Web Wallet" />;

*   Click the “Import wallet” button

<OptimizedImage title=" How to Import Wallet in Komodo Web Wallet" src={howToImportWallet3} alt=" How to Import Wallet in Komodo Web Wallet" />;

*   Import wallet:

*   **·** Enter a name for your wallet.

*   **·** Enter your seed phrase. If your seed is not [BIP39 compliant](https://github.com/bitcoin/bips/blob/master/bip-0039.mediawiki), tap the **"Allow custom seed"** checkbox, then enter **“I understand”** and press **“ok”** in the pop-up window to continue with importing your custom seed.

*   Confirm you have read the **End-User License Agreement (EULA)** and **TERMS and CONDITIONS** by activating the checkboxes.

*   Once all inputs are valid, click the “Import” button

<OptimizedImage title=" How to Import Wallet in Komodo Web Wallet" src={howToImportWallet4} alt=" How to Import Wallet in Komodo Web Wallet" />;

*   Enter and confirm your password, then click “Import”

<OptimizedImage title=" How to Import Wallet in Komodo Web Wallet" src={howToImportWallet5} alt=" How to Import Wallet in Komodo Web Wallet" />;

*   Voila! Your wallet has been created.
