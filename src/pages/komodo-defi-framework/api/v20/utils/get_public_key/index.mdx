# get\_public\_key

The `get_public_key` method returns the compressed secp256k1 pubkey corresponding to the user's seed phrase.

## Arguments

| Structure | Type | Description |
| --------- | ---- | ----------- |
| (none)    |      |             |

#### Response

| Structure   | Type   | Description   |
| ----------- | ------ | ------------- |
| public\_key | string | User's pubkey |
