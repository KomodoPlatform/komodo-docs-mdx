# Initialize SIA Activation

## task::enable\_sia::init {{label : 'task::enable_sia::init', tag : 'API-v2'}}

This method starts the task of activating the SIA coin. It returns a task ID that can be used with the `status` and `user_action` methods to monitor and complete the activation process.

### Request Parameters

| Parameter                | Type   | Required | Default | Description                                                                |
| :----------------------- | :----- | :------: | :-----: | :------------------------------------------------------------------------- |
| client\_conf             | object |     âœ“    |    -    | An object containing the SIA client configuration.                         |
| client\_conf.server\_url | string |     âœ“    |    -    | The URL of the SIA public node. e.g., `https://api.siascan.com/wallet/api` |
| client\_conf.password    | string |     âœ“    |    -    | The password for the SIA wallet daemon.                                    |
| tx\_history              | bool   |     âœ—    | `false` | Whether to enable transaction history for the coin.                        |

### Response Parameters

| Parameter | Type    | Description                    |
| :-------- | :------ | :----------------------------- |
| task\_id  | integer | The ID of the activation task. |
