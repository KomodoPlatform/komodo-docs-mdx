# Enable UTXO Task: Init

## task::enable\_utxo::init {{label : 'task::enable_utxo::init', tag : 'API-v2'}}

Use this method for task managed activation of UTXO coins like KMD, LTC, BTC & DOGE. Refer to the [task managed activation overview](/komodo-defi-framework/api/v20/coin_activation/task_managed/) for activation of other coin types.

| ------------------ | ------ | :------: | :--------------------------: | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

| activation\_params | object |     âœ“    |               -              | A standard [ActivationParams](/komodo-defi-framework/api/common_structures/activation/#activation-params) object containing activation configuration parameters for UTXO coins.                                                                                                                          |
| priv\_key\_policy  | object |     âœ—    | `{"type": "ContextPrivKey"}` | Object specifying the policy to access a private key during activation.
