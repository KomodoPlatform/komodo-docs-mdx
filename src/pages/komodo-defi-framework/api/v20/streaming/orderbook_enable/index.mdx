# Komodo DeFi Framework Method: Enable Orderbook Streaming

## stream::orderbook::enable {{label : 'stream::orderbook::enable', tag : 'API-v2'}}

Using this method, you can enable the orderbook stream for a given pair.

### Request Arguments

| Parameter  | Type    | Required | Default | Description                                                                                                                      |
| ---------- | ------- | :------: | :-----: | -------------------------------------------------------------------------------------------------------------------------------- |
| base       | string  |     âœ“    |    -    | Base currency of a pair                                                                                                          |
| client\_id | integer |     âœ“    |    -    | This ID can be used to access data (e.g. via [http://localhost:7783/event-stream?id=1](http://localhost:7783/event-stream?id=1)) |
| rel        | string  |     âœ“    |    -    | Related currency, also known as the "quote currency"                                                                             |

<Note>
  For this method, `base` and `rel` are essentially interchangeable, and inverting the base coin and rel coin in a request will return the same data.
</Note>
