# Clear NFT Database

## non\_fungible\_tokens::clear\_nft\_db {{label : 'non_fungible_tokens::clear_nft_db', tag : 'API-v2'}}

This method clears the NFT database data stored in the local database for selected (or all) networks.

### Request Arguments

| Parameter  | Type    | Required | Default | Description                                                                                                                                             |
| ---------- | ------- | :------: | :-----: | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| chains     | array   |     ‚úó    |   `-`   | List of networks to remove NFT data for. See [NftNetworkEnum](/komodo-defi-framework/api/common_structures/enums/#nft-network-enum) for allowed values. |
| clear\_all | boolean |     ‚úó    | `false` | If `true`, all NFT data for all networks will be purged.                                                                                                |

<Note>
  You can confirm the NFT database has been cleared by querying the `KOMODEFI.db` database file. For more information, refer to the [Query NFT Database Tables](/komodo-defi-framework/api/v20/non_fungible_tokens/query_nft_database_tables/) documentation.
</Note>
