# Validators

The `experimental::staking::query::ongoing_delegations` method returns information about your node's staking. Currently QTUM and tQTUM (test tokens avalable at `https://testnet-faucet.qtum.info/`), along with Cosmos/Tendermint coins are supported.

## Arguments

| Structure | Type   | Description           |
| --------- | ------ | --------------------- |
| coin      | string | the coin being staked |
