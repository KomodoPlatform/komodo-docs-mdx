# 1inch\_v6\_0\_classic\_swap\_tokens

The `1inch_v6_0_classic_swap_tokens` method returns tokens for classic swaps from 1inch API version 6.0.

## Arguments

| Structure       | Type | Description                                                                  |
| --------------- | ---- | ---------------------------------------------------------------------------- |
| chain_id | numeric | Chain id |


## Response

| Structure | Type             | Description                                                                                                                                                                                            |
| --------- | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| dst_amount | rational number | Destination token amount, in coins units. |
| tokens | list of [TokenInfo](/komodo-defi-framework/api/v20/one_inch_v6_0_classic_swap_tokens/#TokenInfo) | Optional. Available tokens for trades. |

### TokenInfo
| Structure | Type             | Description                                                                                                                                                                                            |
| --------- | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| address | string | Token contract address. |
| symbol | string | Token symbol. |
| name | string | Token name. |
| decimals | numeric | Number of digits after decimal point for the token amount. |
| eip2612 | boolean | Is Eip-2612 supported. |
| is_fot | boolean | Is FoT token. |
| logo_uri | string | Token logo uri. |
| tags | list of string | Token tags. |
